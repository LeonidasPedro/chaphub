# ChapHub — Landing

Landing page do Ecossistema de Inovação ChapHub. React 18 + Vite 5.

## Requisitos

- Node 18+
- npm

## Instalar

```bash
npm install
```

## Rodar em dev

```bash
npm run dev
```

Abre em `http://localhost:3000`.

## Build de produção

```bash
npm run build
```

Saída em `dist/`.

## Preview do build

```bash
npm run preview
```

## Estrutura

```
src/
  components/   seções da landing (Nav, Hero, Sobre, ...)
  hooks/        comportamento (fade-in, count-up, partículas, typed)
  data/         conteúdo (câmaras, parceiros, palavras do typed)
  styles/       CSS global
public/
  assets/fonts/ fontes Organetto
  chaphub/      logos do ecossistema
```
